# 🎨 دليل استخدام محرر القوالب

## الوصول السريع

### من لوحة الإدارة:
1. سجّل الدخول كمسؤول
2. من القائمة الجانبية → **🎨 محرر القوالب**
3. اختر القالب المراد تعديله

### الرابط المباشر:
```
http://your-domain/admin/templates/editor
```

---

## 🎯 المميزات الرئيسية

### ✅ دعم صفحات متعددة
- إضافة صفحات جديدة بحجم A4 (21×27 سم)
- حذف الصفحات غير المرغوبة
- التنقل بين الصفحات

### ✅ تحرير تفاعلي
- **سحب وإفلات**: اسحب أي عنصر لتحريكه
- **تغيير الحجم**: اسحب الحواف لتغيير حجم العنصر
- **تعديل النصوص**: انقر على النص المظلل للتعديل المباشر
- **تحريك دقيق**: استخدم الأسهم (↑ ↓ ← →) للتحريك بدقة

### ✅ الحفظ والاستعادة
- **حفظ على السيرفر**: حفظ دائم مع نسخ احتياطية
- **تصدير JSON**: تحميل القالب كملف
- **استيراد JSON**: رفع قالب محفوظ
- **استعادة افتراضي**: إرجاع القالب الأصلي

### ✅ معاينة وطباعة
- معاينة مباشرة قبل الطباعة
- طباعة تلقائية بحجم A4
- دعم كامل للعربية

---

## 📖 خطوات الاستخدام

### 1️⃣ فتح المحرر
```
لوحة الإدارة → 🎨 محرر القوالب → اختر القالب
```

### 2️⃣ تفعيل وضع التعديل
```
انقر على زر "تفعيل وضع التعديل"
```

### 3️⃣ تعديل العناصر

#### تحريك عنصر:
1. انقر على العنصر لتحديده
2. اسحبه إلى المكان المطلوب
3. أو استخدم الأسهم للتحريك الدقيق

#### تغيير حجم عنصر:
1. حدد العنصر
2. اسحب الحواف أو الزوايا
3. العنصر سيتغير حجمه تلقائياً

#### تعديل نص:
1. انقر على النص المظلل (باللون الأصفر)
2. اكتب النص الجديد
3. انقر خارج النص للحفظ

### 4️⃣ إدارة الصفحات

#### إضافة صفحة جديدة:
```
انقر على "➕ إضافة صفحة"
```

#### حذف صفحة:
```
انقر على "➖ حذف الصفحة الحالية"
```
⚠️ ملاحظة: يجب أن يحتوي القالب على صفحة واحدة على الأقل

### 5️⃣ حفظ التعديلات

#### الحفظ على السيرفر:
```
انقر على "💾 حفظ التغييرات"
```
✅ سيتم إنشاء نسخة احتياطية تلقائياً

#### تصدير كملف:
```
انقر على "📥 تصدير JSON"
```
سيتم تحميل ملف JSON يحتوي على جميع التعديلات

#### استيراد من ملف:
```
انقر على "📤 استيراد JSON" → اختر الملف
```

#### استعادة الافتراضي:
```
انقر على "↺ إرجاع الافتراضي"
```
⚠️ تحذير: سيتم حذف جميع التعديلات!

### 6️⃣ المعاينة والطباعة
```
انقر على "👁️ معاينة الطباعة"
```
سيفتح نافذة معاينة الطباعة

## 🎨 القوالب المتاحة

### 1. شهادة سلبية (📄)
قالب طباعة الشهادة السلبية مع معلومات صاحب الملكية

### 2. بطاقة عقارية (🏠)
قالب طباعة البطاقة العقارية مع تفاصيل العقار

---

## ⚙️ الإعدادات المتقدمة

### اختصارات لوحة المفاتيح:
- **←** : تحريك يساراً
- **→** : تحريك يميناً
- **↑** : تحريك للأعلى
- **↓** : تحريك للأسفل
- **Shift + الأسهم** : تحريك أسرع (10 بكسل)
- **Esc** : إغلاق المحرر (مع تأكيد)

### النسخ الاحتياطية:
- يتم إنشاء نسخة احتياطية عند كل حفظ
- يتم الاحتفاظ بآخر 10 نسخ
- النسخ القديمة تُحذف تلقائياً

### مواقع الملفات:
```
storage/app/templates/
├── settings/           # القوالب المحفوظة
├── backups/            # النسخ الاحتياطية
└── defaults/           # القوالب الافتراضية
```

## 🐛 حل المشاكل الشائعة

### المحرر لا يفتح؟
✅ تأكد من تسجيل الدخول كمسؤول
✅ تحقق من صلاحيات الوصول

### لا يحفظ التعديلات؟
✅ تأكد من أن مجلد `storage/app/templates/` قابل للكتابة
✅ افتح console المتصفح للتحقق من الأخطاء

### العناصر لا تتحرك؟
✅ تأكد من تفعيل "وضع التعديل"
✅ انقر على العنصر أولاً لتحديده

### النصوص لا تتعدل؟
✅ فقط النصوص المظللة (باللون الأصفر) قابلة للتعديل
✅ تأكد من أن وضع التعديل مفعّل

## 💡 نصائح مفيدة

1. **احفظ بانتظام**: احفظ تعديلاتك بشكل دوري
2. **استخدم التصدير**: صدّر القالب كنسخة احتياطية خارجية
3. **جرّب المعاينة**: عاين القالب قبل الحفظ النهائي
4. **استخدم الأسهم**: للتحريك الدقيق استخدم الأسهم
5. **Shift للسرعة**: اضغط Shift مع الأسهم للتحريك السريع
