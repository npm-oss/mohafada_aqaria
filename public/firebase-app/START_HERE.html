⚠️ تنبيه مهم

لا يمكن فتح التطبيق مباشرة من المجلد! يجب تشغيل سيرفر محلي أولاً.

📝 الخطوة 1: شغّل السيرفر المحلي

على Windows: انقر مرتين على الملف:

start-server.bat

على Linux/Mac:

chmod +x start-server.sh ./start-server.sh

🌐 الخطوة 2: افتح المتصفح

بعد تشغيل السيرفر، افتح أحد هذه الروابط:

http://localhost:8000/index.html (للزبون) http://localhost:8000/admin/login.html (للأدمن) http://localhost:8000/test-connection.html (اختبار)

🔐 الخطوة 3: أنشئ حساب أدمن

اذهب إلى Firebase Console وأنشئ حساب:

https://console.firebase.google.com → اختر المشروع: project-811172743097267139 → Authentication → Add User
📖 دليل الاستخدام 🔍 اختبار الاتصال

ℹ️ لماذا نحتاج سيرفر محلي؟

  • التطبيق يستخدم JavaScript Modules (ES6)
  • المتصفحات تمنع تحميل Modules من file:// بسبب CORS
  • السيرفر المحلي يحل هذه المشكلة بتوفير http://
  • Firebase يحتاج اتصال إنترنت نشط
