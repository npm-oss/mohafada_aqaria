# المحافظة العقارية - أولاد جلال

موقع إلكتروني للمحافظة العقارية يتيح للمواطنين تقديم طلبات الشهادات والوثائق العقارية.

## 🚀 المميزات

- طلب شهادة سلبية (جديدة / إعادة استخراج)
- طلب البطاقات العقارية (شخصية، أبجدية، حضرية، ريفية)
- طلب مستخرجات العقود
- لوحة تحكم للإدارة
- تصميم حديث ومتجاوب

## 🛠️ التقنيات

- HTML5 / CSS3 / JavaScript
- Firebase (Firestore + Authentication)
- تصميم RTL للغة العربية

## 📦 التثبيت

1. Clone the repository
2. Configure Firebase in `js/firebase-config.js`
3. Deploy to Vercel or any static hosting

## 🔧 Firebase Setup

1. Create a Firebase project
2. Enable Firestore Database
3. Enable Email/Password Authentication
4. Update security rules
