## 📍 1. نظام الطباعة

### المواقع:
- **قائمة الشهادات**: `/admin/certificates` → زر 🖨️ طباعة
- **عرض شهادة**: `/admin/certificates/{id}` → زر 🖨️ طباعة الشهادة
- **قائمة الوثائق**: `/admin/documents` → زر 🖨️ طباعة
- **عرض وثيقة**: `/admin/documents/{id}` → زر 🖨️ طباعة البطاقة

### الميزات:
✅ طباعة فورية بنقرة واحدة
✅ فتح في نافذة جديدة
✅ تنسيق احترافي بحجم A4
✅ دعم كامل للعربية
✅ بيانات ديناميكية من قاعدة البيانات

### كيفية الاستخدام:
```
1. افتح قائمة الشهادات أو الوثائق
2. انقر على زر "🖨️ طباعة"
3. ستفتح نافذة جديدة بالمستند
4. استخدم Ctrl+P للطباعة أو حفظ كـ PDF
```

## 📍 2. محرر القوالب

### الوصول:
- **من القائمة**: لوحة الإدارة → 🎨 محرر القوالب
- **الرابط المباشر**: `/admin/templates/editor`

### القوالب المتاحة:
1. **📄 شهادة سلبية** - قالب طباعة الشهادة السلبية
2. **🏠 بطاقة عقارية** - قالب طباعة البطاقة العقارية

### الميزات:
✅ **صفحات متعددة**: إضافة/حذف صفحات بحجم A4 (21×27 سم)
✅ **سحب وإفلات**: تحريك العناصر بسهولة
✅ **تغيير الحجم**: تعديل أبعاد العناصر
✅ **تعديل النصوص**: تحرير مباشر للنصوص المظللة
✅ **حفظ ذكي**: نسخ احتياطية تلقائية (آخر 10 نسخ)
✅ **تصدير/استيراد**: JSON للمشاركة والنقل
✅ **معاينة**: عرض قبل الطباعة
✅ **استعادة**: إرجاع القالب الافتراضي

### كيفية الاستخدام:

#### الخطوة 1: فتح المحرر
```
لوحة الإدارة → 🎨 محرر القوالب → اختر القالب
```

#### الخطوة 2: تفعيل وضع التعديل
```
انقر على "تفعيل وضع التعديل"
```

#### الخطوة 3: التعديل

**تحريك عنصر:**
- انقر على العنصر لتحديده
- اسحبه إلى المكان المطلوب
- أو استخدم الأسهم (↑ ↓ ← →) للتحريك الدقيق
- Shift + الأسهم = تحريك أسرع (10 بكسل)

**تغيير حجم عنصر:**
- حدد العنصر
- اسحب الحواف أو الزوايا

**تعديل نص:**
- انقر على النص المظلل (باللون الأصفر)
- اكتب النص الجديد
- انقر خارج النص للحفظ

#### الخطوة 4: إدارة الصفحات

**إضافة صفحة:**
```
انقر على "➕ إضافة صفحة"
```

**حذف صفحة:**
```
انقر على "➖ حذف الصفحة الحالية"
```
⚠️ ملاحظة: يجب أن يحتوي القالب على صفحة واحدة على الأقل

#### الخطوة 5: الحفظ

**حفظ على السيرفر:**
```
انقر على "💾 حفظ التغييرات"
```
✅ سيتم إنشاء نسخة احتياطية تلقائياً

**تصدير كملف:**
```
انقر على "📥 تصدير JSON"
```

**استيراد من ملف:**
```
انقر على "📤 استيراد JSON" → اختر الملف
```

**استعادة الافتراضي:**
```
انقر على "↺ إرجاع الافتراضي"
```
⚠️ تحذير: سيتم حذف جميع التعديلات!

#### الخطوة 6: المعاينة
```
انقر على "👁️ معاينة الطباعة"
```

### اختصارات لوحة المفاتيح:
- **←** : تحريك يساراً
- **→** : تحريك يميناً
- **↑** : تحريك للأعلى
- **↓** : تحريك للأسفل
- **Shift + الأسهم** : تحريك أسرع (10 بكسل)
- **Esc** : إغلاق المحرر (مع تأكيد)

## 📍 3. إدارة الطلبات

### الأزرار المتاحة:

#### في القوائم:
- **👁️ عرض** (أزرق) - عرض تفاصيل الطلب
- **🖨️ طباعة** (أخضر) - طباعة الشهادة/البطاقة
- **✅ موافقة** (أخضر) - الموافقة على الطلب
- **❌ رفض** (أحمر) - رفض الطلب

#### في صفحات العرض:
- **⚙️ معالجة** (أزرق) - معالجة الطلب
- **🖨️ طباعة** (أخضر) - طباعة المستند
- **🗑️ حذف** (أحمر) - حذف الطلب
- **← رجوع** (رمادي) - العودة للقائمة

## 🔗 المسارات الكاملة:

### الطباعة:
```php
GET /admin/certificates/{id}/print  → طباعة شهادة سلبية
GET /admin/documents/{id}/print     → طباعة بطاقة عقارية
```

### محرر القوالب:
```php
GET  /admin/templates/editor                → صفحة القوالب
GET  /admin/templates/editor-frame/{type}   → المحرر
POST /admin/templates/save-settings         → حفظ
GET  /admin/templates/load-settings/{type}  → تحميل
POST /admin/templates/restore/{type}        → استعادة
```

### إدارة الشهادات:
```php
GET    /admin/certificates              → القائمة
GET    /admin/certificates/{id}         → العرض
GET    /admin/certificates/{id}/process → المعالجة
POST   /admin/certificates/{id}/approve → الموافقة
POST   /admin/certificates/{id}/reject  → الرفض
```

### إدارة الوثائق:
```php
GET    /admin/documents                 → القائمة
GET    /admin/documents/{id}            → العرض
GET    /admin/documents/{id}/process    → المعالجة
POST   /admin/documents/{id}/approve    → الموافقة
POST   /admin/documents/{id}/reject     → الرفض
DELETE /admin/documents/{id}            → الحذف
```

## 🐛 حل المشاكل الشائعة:

### المحرر لا يفتح؟
✅ تأكد من تسجيل الدخول كمسؤول
✅ تحقق من صلاحيات الوصول
✅ افتح Console (F12) للتحقق من الأخطاء

### لا يحفظ التعديلات؟
✅ تأكد من أن مجلد `storage/app/templates/` قابل للكتابة
```bash
chmod -R 775 storage/app/templates/
```

### أزرار الطباعة لا تظهر؟
✅ تحقق من أنك في الصفحة الصحيحة
✅ امسح الكاش: `php artisan cache:clear`
✅ امسح الـ view cache: `php artisan view:clear`

### الطباعة لا تعمل؟
✅ تحقق من أن المسارات صحيحة
✅ تأكد من وجود البيانات في قاعدة البيانات
✅ افتح Network tab (F12) لرؤية الأخطاء
