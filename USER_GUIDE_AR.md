# دليل استخدام محرر القوالب 📝

## مقدمة
محرر القوالب يسمح لك بتخصيص شكل ومحتوى الشهادات والبطاقات العقارية.

## الوصول للمحرر

### من لوحة التحكم
1. افتح لوحة التحكم الإدارية
2. اذهب إلى قسم "القوالب"
3. اختر القالب الذي تريد تعديله:
   - 📄 شهادة سلبية
   - 🏠 بطاقة عقارية

## واجهة المحرر

### شريط الأدوات العلوي

#### المجموعة الأولى - التحرير
- **🎨 تفعيل وضع التعديل**: لبدء تعديل القالب
- **💾 حفظ التغييرات**: لحفظ التعديلات على السيرفر
- **↺ إرجاع الافتراضي**: لاستعادة القالب الأصلي

#### المجموعة الثانية - الصفحات
- **➕ إضافة صفحة**: لإضافة صفحة جديدة
- **➖ حذف الصفحة**: لحذف الصفحة الحالية
- **صفحة X من Y**: عداد الصفحات

#### المجموعة الثالثة - التصدير
- **📥 تصدير JSON**: لحفظ القالب كملف
- **📤 استيراد JSON**: لتحميل قالب محفوظ
- **👁️ معاينة الطباعة**: لمعاينة قبل الطباعة

## كيفية الاستخدام

### 1️⃣ تفعيل وضع التعديل
```
1. اضغط على زر "تفعيل وضع التعديل"
2. ستظهر حدود زرقاء حول العناصر
3. النصوص المظللة بالأصفر قابلة للتعديل
```

### 2️⃣ تحريك العناصر
```
1. انقر على العنصر لتحديده
2. اسحبه إلى المكان المطلوب
3. استخدم الأسهم للتحريك الدقيق:
   - ← → ↑ ↓ : تحريك بكسل واحد
   - Shift + أسهم : تحريك 10 بكسل
```

### 3️⃣ تغيير حجم العناصر
```
1. حدد العنصر
2. اسحب الحواف أو الزوايا
3. يمكنك تكبير أو تصغير العنصر
```

### 4️⃣ تعديل النصوص
```
1. انقر مرتين على النص
2. اكتب النص الجديد
3. النصوص المظللة بالأصفر قابلة للتعديل
```

### 5️⃣ حفظ التغييرات
```
1. اضغط على "💾 حفظ التغييرات"
2. انتظر رسالة التأكيد
3. التغييرات محفوظة على السيرفر
```

## المميزات المتقدمة

### إضافة صفحات متعددة
```
1. اضغط "➕ إضافة صفحة"
2. ستظهر صفحة جديدة أسفل الحالية
3. يمكنك تعديلها بنفس الطريقة
```

### تصدير القالب
```
1. اضغط "📥 تصدير JSON"
2. سيتم تحميل ملف .json
3. احتفظ به كنسخة احتياطية
```

### استيراد قالب
```
1. اضغط "📤 استيراد JSON"
2. اختر ملف .json محفوظ
3. سيتم تطبيق القالب فوراً
```

### معاينة الطباعة
```
1. اضغط "👁️ معاينة الطباعة"
2. ستفتح نافذة المعاينة
3. يمكنك الطباعة أو حفظ PDF
```

## نصائح مهمة ⚠️

### ✅ افعل
- احفظ التغييرات بانتظام
- صدّر نسخة احتياطية قبل التعديلات الكبيرة
- استخدم "معاينة الطباعة" للتحقق من النتيجة
- جرب التعديلات على نسخة اختبار أولاً

### ❌ لا تفعل
- لا تغلق الصفحة قبل الحفظ
- لا تعدل عدة قوالب في نفس الوقت
- لا تحذف العناصر الأساسية (العنوان، الختم)
- لا تستخدم خطوط غير مدعومة

## حل المشاكل الشائعة

### المشكلة: الصفحة بيضاء
```
الحل:
1. أعد تحميل الصفحة (F5)
2. امسح الكاش (Ctrl + Shift + R)
3. تحقق من اتصال الإنترنت
4. جرب متصفح آخر
```

### المشكلة: لا يمكن تحريك العناصر
```
الحل:
1. تأكد من تفعيل "وضع التعديل"
2. انقر على العنصر أولاً
3. تأكد من عدم تحديد نص داخل العنصر
```

### المشكلة: التغييرات لا تُحفظ
```
الحل:
1. تحقق من اتصال الإنترنت
2. تأكد من صلاحيات الحفظ
3. جرب تصدير JSON كنسخة احتياطية
4. تواصل مع الدعم الفني
```

### المشكلة: الطباعة غير صحيحة
```
الحل:
1. استخدم "معاينة الطباعة" أولاً
2. تأكد من إعدادات الطابعة (A4)
3. اختر "طباعة الخلفية" في الإعدادات
4. جرب حفظ PDF ثم الطباعة
```

## اختصارات لوحة المفاتيح ⌨️

| الاختصار | الوظيفة |
|---------|---------|
| `←` `→` `↑` `↓` | تحريك العنصر المحدد |
| `Shift + أسهم` | تحريك سريع (10 بكسل) |
| `Ctrl + S` | حفظ (قريباً) |
| `Ctrl + Z` | تراجع (قريباً) |
| `Delete` | حذف العنصر (قريباً) |

## أمثلة عملية

### مثال 1: تغيير عنوان الشهادة
```
1. فعّل وضع التعديل
2. انقر مرتين على "شهادة سلبية"
3. اكتب العنوان الجديد
4. احفظ التغييرات
```

### مثال 2: إضافة حقل جديد
```
1. فعّل وضع التعديل
2. انسخ حقل موجود (Ctrl + C)
3. الصق في مكان جديد (Ctrl + V)
4. عدّل النص
5. احفظ
```

### مثال 3: تغيير موضع الختم
```
1. فعّل وضع التعديل
2. حدد عنصر الختم
3. اسحبه للمكان المطلوب
4. استخدم الأسهم للضبط الدقيق
5. احفظ
```

## الدعم الفني 📞

إذا واجهت مشكلة:
1. افتح Developer Console (F12)
2. التقط صورة للشاشة
3. سجل الخطوات التي أدت للمشكلة
4. تواصل مع الدعم الفني
