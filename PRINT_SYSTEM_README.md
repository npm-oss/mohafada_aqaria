# نظام الطباعة ومحرر القوالب - Firebase/Vercel

## 🎯 الحل المطبق

تم إنشاء نظام طباعة ومحرر قوالب يعمل بالكامل مع Firebase وVercel، بدون الحاجة لـ Laravel backend.

## 📁 الملفات المضافة

### ملفات الطباعة:
- `public/firebase-app/admin/print-certificate.html` - طباعة الشهادات السلبية
- `public/firebase-app/admin/print-document.html` - طباعة البطاقات العقارية

### محرر القوالب:
- `public/firebase-app/admin/template-editor.html` - محرر القوالب التفاعلي

### ملف الاختبار:
- `public/firebase-app/admin/test-print.html` - صفحة اختبار جميع الوظائف

## ✅ الوظائف المتاحة

### 1. أزرار الطباعة
- **تظهر تلقائياً** في صفحات الإدارة للطلبات المعتمدة
- **الشهادات السلبية**: زر طباعة يظهر عند الحالة `approved` أو `extracted`
- **البطاقات العقارية**: زر طباعة يظهر عند الحالة `approved`
- **فتح في نافذة جديدة** للطباعة المباشرة

### 2. محرر القوالب
- **محرر HTML/CSS** تفاعلي
- **معاينة مباشرة** للتغييرات
- **حفظ محلي** في المتصفح
- **استعادة القوالب الافتراضية**
- **قوالب منفصلة** للشهادات والبطاقات

### 3. أزرار الموافقة/الرفض
- **تعمل مع Firebase** مباشرة
- **تظهر فقط للطلبات المعلقة** (`pending`)
- **تحديث فوري** للحالة

## 🔗 الروابط الجديدة

### في واجهة الإدارة:
```
الشهادات السلبية: certificates.html
البطاقات العقارية: documents.html
محرر القوالب: template-editor.html
```

### روابط الطباعة:
```
طباعة شهادة: print-certificate.html?id=CERTIFICATE_ID
طباعة بطاقة: print-document.html?id=DOCUMENT_ID
```

## 🎨 كيفية استخدام محرر القوالب

1. **فتح المحرر**: اضغط على "محرر القوالب" في القائمة الجانبية
2. **اختيار القالب**: اختر "شهادة سلبية" أو "بطاقة عقارية"
3. **التعديل**: 
   - تبويب HTML: تعديل هيكل المحتوى
   - تبويب CSS: تعديل التنسيق والألوان
   - تبويب معاينة: مشاهدة النتيجة
4. **الحفظ**: اضغط "حفظ" لحفظ التعديلات
5. **الاستعادة**: اضغط "استعادة الافتراضي" للعودة للقالب الأصلي

## 🔒 الأمان

- **البيانات محفوظة في Firebase** (آمنة)
- **القوالب محفوظة محلياً** في المتصفح
- **لا توجد معلومات حساسة** في الكود
- **التحقق من الصلاحيات** عبر Firebase Auth

## 🆘 استكشاف الأخطاء

### إذا لم تظهر أزرار الطباعة:
1. تأكد من أن حالة الطلب `approved`
2. تحديث الصفحة
3. تحقق من وحدة تحكم المتصفح للأخطاء

### إذا لم يعمل محرر القوالب:
1. تأكد من تسجيل الدخول
2. امسح cache المتصفح
3. جرب متصفح آخر

### إذا لم تعمل الطباعة:
1. تأكد من وجود البيانات في Firebase
2. تحقق من معرف الطلب في الرابط
3. جرب الطباعة من متصفح آخر
