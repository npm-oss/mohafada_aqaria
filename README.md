# Mohafada Aqaria (Land Registry Admin)

A professional web application for managing land registry documents, negative certificates, and property cards.

## Features
- **Advanced Template Editor**: Design certificates and documents with a Canva-like interface.
- **Auto-Save & Cloud Storage**: Real-time saving to Firebase with local storage fallback.
- **Dynamic Printing**: Generate high-quality PDFs and printouts directly from templates.
- **Admin Dashboard**: Manage user requests and contact messages efficiently.
- **Responsive Design**: Fully functional on desktop and mobile devices.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS, Javascript (ES Modules).
- **Backend**: Firebase Realtime Database & Auth.
- **Libraries**: Interact.js (drag & drop), html2canvas/jsPDF (PDF export).
- **Deployment**: Configured for Vercel.

## Setup
1. Clone the repository.
2. Update `firebase-config.js` with your Firebase credentials.
3. Deploy to Vercel or run locally using any web server.
